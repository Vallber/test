1. скачати MySQL Worckbench
2. встановити бібліотеку pymysql
3. встановити бібліотеку Flask
4. ввсети дані від бази даних в полях user, password, db(host не рухати, томущо Docker працює тільки з хостом db)
5. створити та запустити контейнери через команду docker-compose up --build
6. почекати коли консоль напише "app-1  | MySQL is up - executing command"
7. відкрити сайт написавши в браузері localhost:8000 або як на відео
8. Користуватись
